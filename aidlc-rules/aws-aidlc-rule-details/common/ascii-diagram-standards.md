# ASCII 다이어그램 표준

## 필수: 기본 ASCII만 사용

**중요**: 다이어그램에는 항상 기본 ASCII 문자를 사용합니다(최대 호환성).

### ✅ 허용: `+` `-` `|` `^` `v` `<` `>` 및 영숫자 텍스트

### ❌ 금지: 유니코드 박스 그리기 문자
- 사용 금지: `┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 이유: 글꼴·플랫폼마다 렌더링이 일관되지 않음

## 표준 ASCII 다이어그램 패턴

### 중요: 문자 너비 규칙
**박스 안의 모든 줄은 공백 포함 문자 수가 정확히 같아야 합니다**

✅ 올바름(모든 줄 = 67자):
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                        |
+---------------------------------------------------------------+
```

❌ 잘못됨(너비 불일치):
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                   |
+---------------------------------------------------------------+
```

### 박스 패턴
```
+-----------------------------------------------------+
|                                                     |
|              Calculator Application                 |
|                                                     |
|  Provides basic arithmetic operations for users     |
|  through a web-based interface                      |
|                                                     |
+-----------------------------------------------------+
```

### 중첩 박스
```
+-------------------------------------------------------+
|              Web Server (PHP Runtime)                 |
|  +-------------------------------------------------+  |
|  |  index.php (Monolithic Application)             |  |
|  |  +-------------------------------------------+  |  |
|  |  |  HTML Template (View Layer)               |  |  |
|  |  |  - Form rendering                         |  |  |
|  |  |  - Result display                         |  |  |
|  |  +-------------------------------------------+  |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
```

```
+----------+
     |
     | HTTP POST

+----------+
```

### 수평 흐름
```

```

## 검증

다이어그램을 만들기 전에:
- [ ] 기본 ASCII만: `+` `-` `|` `^` `v` `<` `>`
- [ ] 유니코드 박스 그리기 없음
- [ ] 정렬에는 공백(탭 아님)
- [ ] 모서리는 `+`
- [ ] **모든 박스 줄의 문자 너비 동일**(공백 포함 문자 수)
- [ ] 테스트: 등폭 글꼴에서 모서리가 수직으로 맞는지 확인

## 대안

복잡한 다이어그램은 Mermaid를 사용합니다(`content-validation.md` 참고)
